# collect_all_even

# 题目描述

1. 请在 ```src/collect-all-even.js ``` 中实现 ```collectAllEven```

2. 函数作用：返回给定数组中的所有的偶数  

3. 例如：当传入 `[1, 2, 3, 4, 5,]`， 则函数应该返回 `[2, 4]`
